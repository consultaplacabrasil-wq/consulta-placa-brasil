2.2 API Brasil FREE TIER

URL: apibrasil.com.br | Dashboard: app.apibrasil.io

Dados fornecidos:

  • Marca, modelo, versao, ano, chassi, combustivel, preco FIPE
  • Cobertura: ~92% da frota nacional

Precos:

  • Gratis: 100 consultas/dia (renovacao automatica)
  • Planos premium para maior volume (contato)

Como acessar:

  1. Cadastro em app.apibrasil.io
  2. Ir em "Minhas APIs" > "API Placa Dados"
  3. Copiar DeviceToken e BearerToken

Endpoint:

POST https://gateway.apibrasil.io/api/v2/vehicles/dados

SDK Node.js:

npm i apigratis-sdk-nodejs

Documentacao:

sinesp-v3.apibrasil.com.br | apibrasil.blog

Limitacao: Placas recem-emitidas e veiculos militares podem nao ter dados.

2.12 SERPRO / WSDenatran (API Oficial do Governo) FONTE OFICIAL

URL: gov.br/conecta/catalogo/apis/wsdenatran | loja.serpro.gov.br

Dados fornecidos:

  • 52 tipos de consulta em 3 categorias:
  • Veiculos (RENAVAM): registro, roubo, restricoes judiciais, recall, proprietario
  • Infracoes (RENAINF): multas, status de pagamento
  • Condutores (RENACH): dados da CNH, restricoes

Como acessar:

  1. Obter autorizacao do DENATRAN
  2. Contratar servicos via SERPRO
  3. Requer certificado digital (CNPJ)
  4. Empresas privadas podem acessar via Portaria DENATRAN 15/2016

Precos:

Tabelados por volume (Portaria Senatran 215/2018). Ambiente de homologacao gratuito.

Documentacao:

centraldeajuda.serpro.gov.br/consultasenatran/documentacao

Confiabilidade: A MAIS ALTA - fonte oficial do governo. Disponibilidade 24/7. Criptografia SSL. Processo burocratico para obter acesso.

4. Como Conseguir Acesso aos Dados Veiculares

Caminho 1: Credenciamento direto no SENATRAN

  • Requer CNPJ ativo
  • Justificativa formal de uso
  • Contrato com o SERPRO
  • Certificado digital
  • Exemplo: E o que a Zapay fez (Portarias 750/2018 e 62/2023)
  • Prós: Dados oficiais, maior confiabilidade
  • Contras: Processo burocratico e demorado

Caminho 2: APIs de agregadores privados

  • Empresas como Consultar Placa, API Brasil, FIPE API, Infosimples revendem dados consolidados via API REST
  • Prós: Rapido para integrar, sem burocracia
  • Contras: Custo por consulta, dependencia do provedor
  • Melhor opcao para comecar rapidamente

Caminho 3: Parceria B2B

  • AnyCar oferece API para parceiros
  • Olho no Carro usa Hexagon Automotive
  • Checktudo oferece integracao com dashboard
  • Prós: Negociacao de precos, suporte dedicado
  • Contras: Requer volume minimo, contrato
